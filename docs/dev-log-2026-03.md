## 技術的な学び

### ffmpegによる動画編集パイプライン
```
フレーム抽出 → 内容分析 → カットリストCSV → セグメント切り出し → concat
```
- `ffmpeg -nostdin` でwhileループ内のstdin消費を防止
- tus uploadでは `AuthorizationSignature` + `LibraryId` ヘッダーが必須

### Whisper日本語字幕の最適設定
```bash
whisper audio.wav --model small --language ja \
  --condition_on_previous_text False \
  --no_speech_threshold 0.5 \
  --initial_prompt "トレーニング指導の会話..."
```
- `condition_on_previous_text False` がハルシネーション防止の鍵
- largeモデルはGPUなしでは非現実的（13分音声で40分以上）

### Bunny.net Stream API
- Library API Key + Library ID (数値) が必要
- 大ファイルは tus プロトコル（POST create → PATCH chunks）
- CDNホスト名 `vz-{hex}.b-cdn.net` からLibrary IDは推定不可
