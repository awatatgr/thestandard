- **Standard**: ISTQB UAT / IEEE 829

## Step 1: User Stories & Acceptance Criteria

### US-01: 動画一覧を閲覧する (P0)
> As a viewer, I want to browse available videos, so that I can find content to watch.

**Acceptance Criteria:**
1. ヒーローセクションにサムネイル・タイトル・カテゴリバッジ・再生時間・アングル数が表示される
2. 動画カードがグリッド表示され、クリックで詳細ページに遷移する
3. カテゴリフィルターで動画を絞り込める

### US-02: 動画を再生する (P0)
> As a viewer, I want to play videos with multi-angle sync, so that I can study movements from different perspectives.

**Acceptance Criteria:**
1. 再生・一時停止・シーク・スキップが動作する
2. マルチアングルが同期再生される（0.15s以内）
3. レイアウト切替（シングル/均等/メイン+サブ）で再生位置が維持される

### US-03: チャプターでナビゲートする (P1)
> As a viewer, I want to jump to specific exercises via chapters, so that I can practice particular movements.

**Acceptance Criteria:**
1. サイドバーにチャプター一覧が表示され、クリックでシークする
2. アクティブチャプターがハイライトされる
3. エクササイズオーバーレイに進捗ドットが表示され、クリック可能

### US-04: キーボード・タッチ操作で制御する (P1)
> As a viewer, I want to control playback with keyboard shortcuts and touch gestures, so that I can navigate efficiently.

**Acceptance Criteria:**
1. Space/K で再生/一時停止、J/L で ±10s スキップ
2. モバイルでダブルタップ ±10s が動作する
3. input要素フォーカス中はショートカット無効

### US-05: 字幕を表示する (P1)
> As a viewer, I want to toggle subtitles, so that I can follow spoken instructions.

**Acceptance Criteria:**
1. 字幕対応動画で CC ボタンが表示される
2. ON/OFF切替で字幕テキストが表示/非表示される
