# UAT テスト定義 — 管理者ロール (Admin)

| 項目 | 値 |
|------|-----|
| プロダクト | THE STANDARD (マルチアングル動画閲覧プラットフォーム) |
| テスト対象URL | https://thestandard.coach/admin |
| ロール | 管理者 (admin権限) |
| バージョン | v1.0 |
| 作成日 | 2026-03-19 |
| 準拠規格 | ISTQB CTFL / IEEE 829-2008 |

---

## テスト環境・前提条件

| 項目 | 内容 |
|------|------|
| 認証 | Supabase マジックリンク + profiles.role = "admin" |
| API認証 | Bearer token (sessionStorage: `thestandard_admin_token`) |
| バックエンド | Cloudflare Pages Functions + KV |
| テストデータ (初期状態) | 動画7本 / 総時間44分 / カテゴリ3 / 公開5本・下書き2本 |
| サンプル動画 | stretch-full (published), squat-lunge (draft) |
| カテゴリ | training, drill, method, interview |
| ステータス | draft / published / archived |

## Step 1: ユースケース (ユーザーストーリー)

| # | ユーザーストーリー | 受入基準 | 優先度 |
|---|-------------------|---------|--------|
| US-01 | 管理者として、ダッシュボードでKPIを確認したい。コンテンツの全体像を把握するため。 | 動画数・総時間・カテゴリ数・公開中の4指標がリアルタイムに表示される | P0 |
| US-02 | 管理者として、動画一覧を閲覧・管理したい。全動画の状態を一目で把握するため。 | 全ステータスの動画がサムネ・バッジ付きで一覧表示される | P0 |
| US-03 | 管理者として、動画を検索・フィルターしたい。目的の動画を素早く見つけるため。 | テキスト検索 + カテゴリ + ステータスで絞り込める | P1 |
| US-04 | 管理者として、新規動画を登録したい。撮影した動画をプラットフォームに追加するため。 | ID・タイトル・カテゴリ・アングルを入力して作成でき、一覧に反映される | P0 |
| US-05 | 管理者として、既存動画を編集したい。メタデータの修正やアングル・チャプターの追加をするため。 | フォームに既存データがプリフィルされ、更新後にAPIに反映される | P1 |
| US-06 | 管理者として、動画を公開/非公開に切り替えたい。視聴者への公開タイミングを制御するため。 | ワンクリックでステータスがトグルし、視聴者側の表示に即時反映される | P0 |
| US-07 | 管理者として、不要な動画を削除したい。コンテンツを整理するため。 | 確認ダイアログの後に削除され、APIからも取得不可になる | P1 |
| US-08 | 管理者として、動画のEmbed URLを共有したい。外部サイトに埋め込むため。 | URLをコピーでき、認証なしでiframe埋め込みが動作する | P2 |
| US-09 | 管理者として、Ingest Appに接続設定を渡したい。撮影ワークフローを自動化するため。 | 接続設定JSON(api_endpoint, admin_token, bunny_cdn_hostname)がクリップボードにコピーされる | P2 |
| US-10 | 管理者として、admin権限のないユーザーが管理画面にアクセスできないことを確認したい。セキュリティ確保のため。 | 非admin/未認証ユーザーはリダイレクトされる | P0 |

## テスト実行順序 (推奨)

依存関係に基づく推奨実行順序:

1. **認証・認可ガード** (ADMIN-001 ~ 005) — 全テストの前提条件
2. **ダッシュボード KPI** (ADMIN-010 ~ 017) — 既存データの確認
3. **動画一覧管理** (ADMIN-020 ~ 026) — 一覧表示の確認
4. **検索・フィルター** (ADMIN-030 ~ 038) — 一覧操作
5. **新規動画作成** (ADMIN-040 ~ 051) — test-video-001 を作成 (後続テストで使用)
6. **動画編集** (ADMIN-060 ~ 069) — 既存動画の更新
7. **公開・非公開切替** (ADMIN-070 ~ 075) — ステータス操作
8. **Embed共有** (ADMIN-090 ~ 098) — 外部共有機能
9. **削除** (ADMIN-080 ~ 085) — test-video-001 を削除 (データ破壊的操作は最後)
10. **Ingest App接続設定** (ADMIN-100 ~ 105) — 独立機能、任意タイミング
