# B. 管理者向け UAT

## UAT-B1: 動画の新規登録

**優先度**: P0
**E2Eカバー**: `admin-create-video.spec.ts` [自動+手動]

### 前提条件
- 管理者としてログイン済み（Admin権限）
- Bunny.net に動画がアップロード済み（Stream ID が分かる状態）

### テスト手順と期待結果

| # | 手順 | 期待結果 |
|---|------|---------|
| 1 | `/admin/videos/new` にアクセス | - [ ] 「新規動画を追加」タイトルが表示<br>- [ ] 「基本情報」セクション表示 |
| 2 | 基本情報を入力 | - [ ] ID欄: プレースホルダー「stretch-full」<br>- [ ] タイトル欄: 必須マーク（*）付き<br>- [ ] カテゴリ: デフォルト「トレーニング」、4択 |
| 3 | アングルを設定（デフォルト1つ） | - [ ] 「アングル (1)」ヘッダー表示<br>- [ ] ID、ラベル（デフォルト「正面」）、Bunny Stream ID、字幕URL の4フィールド |
| 4 | 「追加」ボタンでアングルを増やす | - [ ] 「アングル (2)」に更新<br>- [ ] 2つ目のアングルフォームが表示 |
| 5 | チャプターを追加 | - [ ] 「追加」ボタンでチャプター入力欄が出現<br>- [ ] 名前、開始(秒)、終了(秒) の3フィールド |
| 6 | 「作成」ボタンをクリック | - [ ] 「動画を作成しました」トースト（緑）が3秒間表示 |
| 7 | 自動遷移を確認 | - [ ] `/admin/videos` に遷移<br>- [ ] 作成した動画が一覧に表示される |
| 8 | 視聴者側で確認（下書きの場合はAPIのみ） | - [ ] `GET /api/videos/{id}` で取得可能<br>- [ ] アングル・チャプターが正しく保存されている |

### エッジケース
- [ ] ID・タイトル未入力で「作成」→ 「ID・タイトル・アングルは必須です」エラートースト
- [ ] 重複IDで作成 → APIエラー（409）が表示される
- [ ] 「キャンセル」ボタンで `/admin/videos` に戻る（データ未保存）
- [ ] アングル1つの状態では削除ボタン（X）が非表示

## UAT-B2: 動画の公開/非公開切替

**優先度**: P0
**E2Eカバー**: `admin-videos.spec.ts` [自動+手動]

### 前提条件
- 管理者ログイン済み
- draft 状態の動画が少なくとも1つ存在

### テスト手順と期待結果

| # | 手順 | 期待結果 |
|---|------|---------|
| 1 | `/admin/videos` にアクセス | - [ ] 動画リストが表示<br>- [ ] 各動画にステータスバッジ（「下書き」or「公開」）表示 |
| 2 | draft動画の目アイコン（公開ボタン）をクリック | - [ ] 「公開しました」トースト表示<br>- [ ] ステータスバッジが「公開」に変化 |
| 3 | 視聴者側 (`/`) で確認 | - [ ] 公開した動画が一覧に表示される |
| 4 | 管理画面に戻り、同じ動画の目アイコン（非公開ボタン）をクリック | - [ ] 「非公開にしました」トースト表示<br>- [ ] ステータスバッジが「下書き」に変化 |
| 5 | 視聴者側で再確認 | - [ ] 非公開にした動画が一覧から消える |

### エッジケース
- [ ] API接続不可時は「APIに接続できません。静的データを表示中です。」警告
- [ ] 静的データ表示時は公開/非公開切替が機能しない（API必須）

**優先度**: P2
**E2Eカバー**: `admin-videos.spec.ts` (コピーボタンのみ) [手動中心]

### 前提条件
- 管理者ログイン済み
- 公開済み動画が存在

### テスト手順と期待結果

| # | 手順 | 期待結果 |
|---|------|---------|
| 1 | `/admin/videos` で動画のコピーアイコンをクリック | - [ ] 「Embed URLをコピーしました」トースト表示<br>- [ ] クリップボードにURLが入る |
| 2 | コピーしたURLをブラウザで開く | - [ ] `/embed/{id}` が表示<br>- [ ] リッチモードで動画プレイヤーが動作 |
| 3 | 編集ページのEmbedセクションを確認 | - [ ] Embed URLが表示（コピーボタン付き）<br>- [ ] iframe snippetが表示（コピーボタン付き） |

### エッジケース
- [ ] `?controls=minimal` を追加するとUIなしのミニマルモード
