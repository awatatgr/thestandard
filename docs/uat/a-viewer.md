## UAT-A1: ログイン〜動画視聴の基本フロー

**優先度**: P0
**E2Eカバー**: `login.spec.ts` + `video-list.spec.ts` [自動+手動]

### 前提条件
- Supabase にユーザーアカウントが登録済み
- 未ログイン状態（sessionStorage クリア済み）

### テスト手順と期待結果

| # | 手順 | 期待結果 |
|---|------|---------|
| 1 | ブラウザで `/` にアクセス | - [ ] ログインページ (`/login`) にリダイレクトされる |
| 2 | 「THE STANDARD」のタイトルと「メールアドレスでログイン」が表示されることを確認 | - [ ] ロゴとサブタイトルが表示される |
| 3 | メールアドレスを入力し「ログインリンクを送信」をクリック | - [ ] ボタンが「送信中...」に変わる |
| 4 | 送信完了を確認 | - [ ] 「メールを確認してください」画面に遷移<br>- [ ] 入力したメールアドレスが表示される |
| 5 | メールボックスでマジックリンクを確認しクリック | - [ ] ブラウザで `/` (動画一覧) が表示される |
| 6 | ヒーローセクションを確認 | - [ ] 最初の動画のタイトル・カテゴリバッジ・所要時間・アングル数が表示<br>- [ ] 「再生する」ボタンがある |
| 7 | ヒーロー or ビデオカードをクリック | - [ ] `/videos/{id}` に遷移<br>- [ ] 動画プレイヤーが表示される |
| 8 | 再生ボタンをクリック | - [ ] 動画が再生開始される |

### エッジケース
- [ ] 無効なメールアドレスで送信 → HTML5バリデーションでブロック
- [ ] 「別のメールアドレスを使う」リンクでフォームに戻れる
- [ ] 既にログイン済みの状態で `/login` にアクセス → `/` にリダイレクト

## UAT-A2: マルチアングル同期再生

**優先度**: P0
**E2Eカバー**: `video-detail.spec.ts` (レイアウト切替のみ) [手動中心]

### 前提条件
- ログイン済み
- 2アングル以上の動画が存在（例: `stretch-full` — 正面+側面）
- 3アングル動画も用意（例: `stretch-3view` — メイン+正面+側面）

### テスト手順と期待結果

| # | 手順 | 期待結果 |
|---|------|---------|
| 1 | 2アングル動画 (`/videos/stretch-full`) にアクセス | - [ ] レイアウト切替バー（Layout表記 + 3ボタン）が表示<br>- [ ] デフォルトで「均等並び」が選択状態 |
| 2 | 均等ビューで2アングルが並んで表示されていることを確認 | - [ ] 正面・側面の2つの動画が横並び<br>- [ ] 各アングルにラベル（「正面」「側面」）バッジ表示 |
| 3 | 「シングルビュー」ボタンをクリック | - [ ] 1つのアングルのみ表示<br>- [ ] 下にアングルタブ（「正面」「側面」）が表示 |
| 4 | 「側面」タブをクリック | - [ ] 表示が側面アングルに切り替わる |
| 5 | 「メイン+サブ」ボタンをクリック | - [ ] メインが大きく、サブが右側に小さく表示<br>- [ ] マスターアングルが大画面側になる |
| 6 | 再生ボタンをクリックして再生開始 | - [ ] 全アングルが同時に再生開始される |
| 7 | 10秒以上再生後、同期ドリフトを確認 | - [ ] 同期ボタンが緑色（ドリフト < 0.1s）<br>- [ ] ドリフト値の数字表示が0.15s以内 |
| 8 | サブアングルの一時停止ボタンをクリック（ホバーで表示） | - [ ] そのアングルのみ一時停止<br>- [ ] 他のアングルは再生継続 |
| 9 | サブアングルのミュートボタンを確認 | - [ ] マスター以外はデフォルトでミュート<br>- [ ] クリックでミュート解除可能 |
| 10 | 再生速度を1.5xに変更 | - [ ] 全アングルが1.5x速度で同期再生される |

### エッジケース
- [ ] 1アングル動画（例: `running-form`）ではレイアウトスイッチャーが非表示
- [ ] 3アングル動画で3画面が正しくグリッド表示される
- [ ] レイアウト切替時に再生位置が維持される
- [ ] 同期ドリフトが0.15sを超えた場合、自動で再同期される（ボタン色が黄色→緑に戻る）

## UAT-A3: チャプターナビゲーション

**優先度**: P0
**E2Eカバー**: `video-detail.spec.ts` + `embed.spec.ts` [自動+手動]

### 前提条件
- ログイン済み
- `exercises` 付き動画が存在（例: `stretch-3view` — 16チャプター）

### テスト手順と期待結果

| # | 手順 | 期待結果 |
|---|------|---------|
| 1 | exercises付き動画 (`/videos/stretch-3view`) にアクセス | - [ ] サイドバーに「チャプター」と「動画一覧」のタブが表示<br>- [ ] 「チャプター」がデフォルト選択 |
| 2 | チャプターリストを確認 | - [ ] 全16チャプターが番号付きで表示<br>- [ ] 各チャプターに名前と開始時間が表示 |
| 3 | 「長座体前屈（ウォームアップ）」チャプターをクリック | - [ ] 動画が該当秒数（17秒）にシーク<br>- [ ] クリックしたチャプターがアクティブ表示（左ボーダー + 白テキスト） |
| 4 | 動画上のオーバーレイを確認 | - [ ] 画面上部にチャプター名が大きく表示<br>- [ ] 右上に進捗バッジ（例: 「2 / 16」） |
| 5 | 画面下部の進捗ドットを確認 | - [ ] 16個のドットが並ぶ<br>- [ ] 現在のチャプターのドットが大きく色付き |
| 6 | 別のドットをクリック | - [ ] 対応するチャプターの開始位置にジャンプ |
| 7 | 「動画一覧」タブに切り替え | - [ ] 全動画のサムネイルリストが表示<br>- [ ] 現在の動画がアクティブ表示（左ボーダー + 再生アイコン） |

### エッジケース
- [ ] チャプターなし動画ではチャプタータブが非表示、「動画一覧」のみ
- [ ] 動画一覧から別の動画をクリック → その動画の詳細ページに遷移
- [ ] サイドバーの開閉トグルボタンが動作する

## UAT-A4: カテゴリフィルタリング

**優先度**: P1
**E2Eカバー**: `video-list.spec.ts` [自動]

### 前提条件
- ログイン済み
- 各カテゴリの動画が存在（training, drill, method）

### テスト手順と期待結果

| # | 手順 | 期待結果 |
|---|------|---------|
| 1 | `/` にアクセス | - [ ] 「すべて」ボタンがアクティブ（primary カラー）<br>- [ ] 全カテゴリの動画が表示 |
| 2 | 「ドリル」ボタンをクリック | - [ ] ドリルカテゴリの動画のみ表示（1件: バーベルスクワット＆ランジ）<br>- [ ] 「ドリル」ボタンがアクティブに変化 |

| 5 | 「すべて」ボタンをクリック | - [ ] 全カテゴリの動画が再表示<br>- [ ] 「すべて」がアクティブに戻る |

### エッジケース

## UAT-A5: キーボードショートカット

**優先度**: P1
**E2Eカバー**: なし [手動のみ]

### 前提条件
- ログイン済み
- 動画詳細ページを開いている

### テスト手順と期待結果

| # | 手順 | 期待結果 |
|---|------|---------|
| 1 | `Space` または `K` キーを押す | - [ ] 再生/一時停止がトグル |
| 2 | `J` キーを押す | - [ ] 10秒巻き戻し |
| 3 | `L` キーを押す | - [ ] 10秒早送り |
| 4 | `←` 矢印キーを押す | - [ ] 5秒巻き戻し |
| 5 | `→` 矢印キーを押す | - [ ] 5秒早送り |
| 6 | `F` キーを押す | - [ ] フルスクリーン切替 |
| 7 | `M` キーを押す | - [ ] ミュート/ミュート解除トグル |
| 8 | 一時停止中に `,` キーを押す | - [ ] 1フレーム戻る<br>動画再生中は `,` 無効 |
| 9 | 一時停止中に `.` キーを押す | - [ ] 1フレーム進む |

### エッジケース
- [ ] フォーカスがテキスト入力欄にある時はショートカットが無効化される
- [ ] マルチビューでもマスタープレイヤーに対してショートカットが有効

## UAT-A7: 字幕ON/OFF

**E2Eカバー**: `video-detail.spec.ts` + `embed.spec.ts` [自動]

- 字幕なし動画も存在（例: `running-form`）

### テスト手順と期待結果

| # | 手順 | 期待結果 |
|---|------|---------|
