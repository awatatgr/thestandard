## 1本あたりの撮影フロー

```
① スレート（声）「ダウンドッグ、テイク1」

② 導入（15-30秒）
   トレーナー:「この種目は○○に効きます。○○な人に特にオススメです。」

③ NG例（30秒）
   一般人がやりがちな間違い
   例:「背中が丸まっている」「膝が伸びきっている」

④ 正しいフォーム実演（1-2分）
   ゆっくりとポーズに入る過程を見せる
   → キープ中に呼吸の仕方も説明

⑤ 質問タイム（1分）
   プロデューサーから:
   ・「今どこが伸びている感覚ですか？」
   ・「呼吸はどのタイミングで吐きますか？」
   ・「痛みが出る場合はどう調整しますか？」
   ・「初心者はどこまでやればいいですか？」

⑥ バリエーション / 軽減法（30秒）
   「硬い人はここにブロックを置いて」
   「膝が痛い人はこうして」

⑦ Bカメ追加ショット（必要に応じて）
   注目部位をクローズアップで再度実演
```

## Aカメ・Bカメ配置ガイド

### Aカメ（iPhone 15 Pro Max） — 固定
- 三脚：胸の高さ
- フレーミング：全身 + 上下左右にマージン
- **ヨガマット全体が収まる画角**
- 基本触らない（全種目共通ポジション）

### Bカメ（iPhone 16 Pro） — 種目ごとに移動

| 種目 | Bカメ位置 | 狙い |
|------|----------|------|
| ダウンドッグ | 側面・低め | ハムストリングの伸び、背中のライン |
| アップドッグ | 側面・低め | 腰椎のカーブ、胸の開き |
| 戦士のポーズ | 正面・膝レベル | 前脚の膝角度、足の向き |
| 三角のポーズ | 正面 | 体側のライン、骨盤の向き |
| 鳩のポーズ | 斜め前・低め | 股関節の角度、臀部 |
| キャット&カウ | 側面・低め | 背骨のカーブの変化 |
| ハムストリング | 側面 | 膝裏、足先 |
| 股関節オープナー | 正面・低め | 内転筋、膝の位置 |
| 胸椎回旋 | 上から or 斜め | 胸椎のひねり、肩甲骨 |
| チャイルドポーズ | 側面・低め | 背中のライン、臀部 |

## ヨガ撮影の注意点

1. **ヨガマットは明るい色を推奨** — 暗いマットは背景と同化しやすい
2. **服装はタイトめ推奨** — 体のラインがわかる方が教材として価値が高い
3. **裸足を映す** — ヨガでは足裏の接地が重要
4. **呼吸の音を拾う** — ピンマイク必須。呼吸のタイミングがコンテンツの一部
5. **静止ポーズのキープ時間** — 最低5秒キープしてからカット
6. **左右両方やる種目** — 必ず左右とも撮影すること（片側だけNG）
7. **トランジション（ポーズ間の移行）** — 流れを見せたい場合はカットしない
